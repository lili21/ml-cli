# ng-cli

A simple CLI for scaffolding napos front-end projects.

# 安装
``` bash
$ cnpm install ng-cli -g
```

# 使用
``` bash
$ ng init [options] <project-name>
```

## 栗子
``` bash
$ ng init melody-food
```
上面的指令会使用默认的模板，按照类型组织，也就是目前melody, rose使用的方式。

``` bash
$ng init -f melody-food
```
这个指令会使用按业务模块组织的模板。
